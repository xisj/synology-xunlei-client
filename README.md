# synology-xunlei-client
给群晖nas的迅雷开发的本地客户端，每次点击或者复制下载链接后都会自动弹出下载框。

下载地址： https://github.com/xisj/synology-xunlei-client/releases

## 功能
1. 打开 nas 下载文件夹
2. 速度球查看下载进度
3. 绑定下载协议，点击后自动激活迅雷下载
4. 退出程序后彻底退出
